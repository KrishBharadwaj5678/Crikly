# 🏏 Crikly

**Crikly** is an interactive web application that visualizes IPL 2024 performance data.
Track the top run-scorers and leading wicket-takers with dynamic and engaging charts.

![CriklyDemo](https://github.com/KrishBharadwaj5678/Crikly/raw/main/CriklyDemo.png)


## ✨ Features

| Feature               | Description                                 |
| --------------------- | ------------------------------------------- |
| 📊 Interactive Charts | Visualize player stats using dynamic charts |
| 🏏 Top Batsmen        | View highest run scorers of IPL 2024        |
| 🎯 Top Bowlers        | Analyze leading wicket-takers               |
| ⚡ Fast & Responsive   | Smooth performance across devices           |
| 🎨 Clean UI           | Minimal and easy-to-understand interface    |
| 🌐 Web-Based          | Runs directly in the browser                |

---

## 🛠️ Tech Stack

| Technology       | Purpose                              |
| ---------------- | ------------------------------------ |
| 🌐 **HTML5**     | Structure of the web application     |
| 🎨 **CSS3**      | Styling and layout                   |
| ⚡ **JavaScript** | Handles logic and data visualization |
| 📊 **Chart.js**  | Creates interactive charts           |
